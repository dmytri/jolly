# Homepage Context

Durable approved homepage product/brand context. `homepage/index.html` is derived from this file.

## Product

Jolly — a homepage + `npx` CLI + agent setup workflow for setting up a Saleor Cloud storefront.

## Target audience

AI agents (primary) and the humans who delegate to them (secondary).

## Core value proposition

Set up a fully operational Saleor Cloud storefront in minutes, with minimal manual steps.

Only three things need the human:
1. New account creation (Saleor Cloud, Vercel, or Stripe if needed)
2. Browser OAuth consent
3. Pasting secret values (Stripe API keys)

## Key action

The primary CTA is a copy-box that gives the visitor a single command to paste into their agent:

> Read https://jolly.cool/setup and follow the instructions to set up Jolly

## Supported environments

- Generic agents (any agent that supports `npx`)
- Zed (agent mode)
- Claude Code
- Cursor (composer agent)
- OpenCode
- Pi.dev

## What the agent handles

1. Connects or creates a Saleor Cloud store
2. Deploys a Paper storefront to Vercel
3. Configures Stripe test checkout
4. Delivers a working store — only account creation and secret keys need the human

## Brand association

Built for Saleor Cloud (https://saleor.io).
